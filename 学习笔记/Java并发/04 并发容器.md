# BlockQueue

> BlockQueue与Queue接口兼容的，同时在基础上增加了同步的功能，BlockQueue接口的方法

```java
public interface BlockingQueue<E> extends Queue<E> {
   
    void put(E e) throws InterruptedException;
    
    boolean add(E e);

    boolean offer(E e);
       
    boolean offer(E e, long timeout, TimeUnit unit)
        throws InterruptedException;

    E take() throws InterruptedException;
    
    E poll(long timeout, TimeUnit unit)
        throws InterruptedException;
}
```

# ArrayBlockingQueue

> 数组结构的环形队列，使用1把锁、2个条件

```java
public class ArrayBlockingQueue<E> extends AbstractQueue<E>
        implements BlockingQueue<E>, java.io.Serializable {

    final Object[] items;

    int takeIndex;
    int putIndex;
    
    int count;

    // 使用了一把锁
    final ReentrantLock lock;

    // 2个条件
    private final Condition notEmpty;
    private final Condition notFull;
}
```



## put方法

```java
    public void put(E e) throws InterruptedException {
        checkNotNull(e);
        final ReentrantLock lock = this.lock;
        lock.lockInterruptibly();
        try {
            while (count == items.length)
                
                // put方法等待非满条件
                notFull.await();
            enqueue(e);
        } finally {
            lock.unlock();
        }
    }

    private void enqueue(E x) {
        
        final Object[] items = this.items;
        
        items[putIndex] = x;
        if (++putIndex == items.length)
            putIndex = 0;
        count++;
        
        //通知非空条件
        notEmpty.signal();
    }
```



## take方法

```java
    public E take() throws InterruptedException {
        
        final ReentrantLock lock = this.lock;
        lock.lockInterruptibly();
        try {
            while (count == 0)
                
                // take方法等待非空条件
                notEmpty.await();
            return dequeue();
        } finally {
            lock.unlock();
        }
    }

    private E dequeue() {
        final Object[] items = this.items;
        
        E x = (E) items[takeIndex];
        items[takeIndex] = null;
        if (++takeIndex == items.length)
            takeIndex = 0;
        count--;
        if (itrs != null)
            itrs.elementDequeued();
        
        //通知非满条件
        notFull.signal();
        return x;
    }
```



# LinkedBlockingQueue

> 链表结构的队列，使用了2把锁、2个条件，控制入队、出队操作，2个操作可以同时进行，使用原子整型记录元素个数

```java
public class LinkedBlockingQueue<E> extends AbstractQueue<E>
        implements BlockingQueue<E>, java.io.Serializable {
    
    private final int capacity;

    private final AtomicInteger count = new AtomicInteger();

    transient Node<E> head;
    private transient Node<E> last;

    /*
     * 使用了2把锁、2个条件，入队、出队可以同时操作，
     * 并不互斥，但对于count变量，是原子类型
     */
    private final ReentrantLock takeLock = new ReentrantLock();
    private final Condition notEmpty = takeLock.newCondition();

    private final ReentrantLock putLock = new ReentrantLock();
    private final Condition notFull = putLock.newCondition();
    
    // 设置容量
    public LinkedBlockingQueue(int capacity) {
        if (capacity <= 0) throw new IllegalArgumentException();
        this.capacity = capacity;
        last = head = new Node<E>(null);
    }
    
	public LinkedBlockingQueue() {
        this(Integer.MAX_VALUE);
    }
}
```



## put方法

> put方法调用putLast方法

```java
	public void put(E e) throws InterruptedException {
        if (e == null) throw new NullPointerException();
       
        int c = -1;
        Node<E> node = new Node<E>(e);
        final ReentrantLock putLock = this.putLock;
        final AtomicInteger count = this.count;
        putLock.lockInterruptibly();
        try {

            while (count.get() == capacity) {
                notFull.await();
            }
            enqueue(node);
            c = count.getAndIncrement();
            if (c + 1 < capacity)
                
                // 入队后，发现队列不满，则通知非满条件
                notFull.signal();
        } finally {
            putLock.unlock();
        }
    
        if (c == 0)
            
            /* 
             * 通知非空条件，通知前需要获得锁，通知后
             * 需要释放锁。
             */
            signalNotEmpty();
    }

    private void signalNotEmpty() {
        final ReentrantLock takeLock = this.takeLock;
        takeLock.lock();
        try {
            notEmpty.signal();
        } finally {
            takeLock.unlock();
        }
    }
```



## take方法

```java
    public E take() throws InterruptedException {
        E x;
        int c = -1;
        final AtomicInteger count = this.count;
        final ReentrantLock takeLock = this.takeLock;
        takeLock.lockInterruptibly();
        try {
            while (count.get() == 0) {
                notEmpty.await();
            }
            x = dequeue();
            c = count.getAndDecrement();
            if (c > 1)
                
                // 还有元素，通知非空条件
                notEmpty.signal();
        } finally {
            takeLock.unlock();
        }
        if (c == capacity)
            
            // 队列未满则通知非满条件
            signalNotFull();
        return x;
    }

    private void signalNotFull() {
        final ReentrantLock putLock = this.putLock;
        putLock.lock();
        try {
            notFull.signal();
        } finally {
            putLock.unlock();
        }
    }
```



# PriorityBlockingQueue

> 优先级同步队列，`元素`需要可以进行比较，需要实现Comparable接口，只有非空条件，入队时，容量不足则执行扩容

```java
public class PriorityBlockingQueue<E> extends AbstractQueue<E>
    implements BlockingQueue<E>, java.io.Serializable {
    
    //使用数组实现小顶堆
	private transient Object[] queue;

    private transient int size;
    
    // 可以传入比较器
    private transient Comparator<? super E> comparator;

    private final ReentrantLock lock;
    private final Condition notEmpty;  
}
```



## put方法

```java
    public void put(E e) {
        offer(e);
    }

	public boolean offer(E e) {
        if (e == null)
            throw new NullPointerException();
        final ReentrantLock lock = this.lock;
        lock.lock();
        int n, cap;
        Object[] array;
        while ((n = size) >= (cap = (array = queue).length))
            
            // 扩容
            tryGrow(array, cap);
        try {
            Comparator<? super E> cmp = comparator;
            if (cmp == null)
                siftUpComparable(n, e, array);
            else
                // 没有比较器则使用元素的自然排序
                siftUpUsingComparator(n, e, array, cmp);
            size = n + 1;
            notEmpty.signal();
        } finally {
            lock.unlock();
        }
        return true;
    }
```



## take方法

```java
    public E take() throws InterruptedException {
        final ReentrantLock lock = this.lock;
        lock.lockInterruptibly();
        E result;
        try {
            while ( (result = dequeue()) == null)
                notEmpty.await();
        } finally {
            lock.unlock();
        }
        return result;
    }

    private E dequeue() {
        int n = size - 1;
        if (n < 0)
            return null;
        else {
            Object[] array = queue;
            
            // 小顶堆第一个元素出队
            E result = (E) array[0];
            E x = (E) array[n];
            array[n] = null;
            Comparator<? super E> cmp = comparator;
            if (cmp == null)
                siftDownComparable(0, x, array, n);
            else
                siftDownUsingComparator(0, x, array, n, cmp);
            size = n;
            return result;
        }
    }
```



# DelayQueue

> 延迟队列，按延迟时间从小到大的优先队列，延迟时间为`未来要执行的时间 - 当前时间`

```java
// 元素需要实现Delayed接口
public class DelayQueue<E extends Delayed> extends AbstractQueue<E>
    implements BlockingQueue<E> {
    
	private final transient ReentrantLock lock = new ReentrantLock();
    private final PriorityQueue<E> q = new PriorityQueue<E>();

    private final Condition available = lock.newCondition();
}
```



## put方法

```java
    public void put(E e) {
        offer(e);
    }

	public boolean offer(E e) {
        final ReentrantLock lock = this.lock;
        lock.lock();
        try {
            
            // 放入优先队列
            q.offer(e);
            
            
            /*
             * 放入元素后，发现该元素延迟时间最小，
             * 则通知其他线程，如果不是，不需要通
             * 知其他线程
             */
            if (q.peek() == e) {
                leader = null;
                
                available.signal();
            }
            return true;
        } finally {
            lock.unlock();
        }
    }
```



## take方法

> 使用了`leader变量`，记录了等待出队元素的线程，只有出队方法会阻塞

```java
    public E take() throws InterruptedException {
        final ReentrantLock lock = this.lock;
        lock.lockInterruptibly();
        try {
            for (;;) {
                E first = q.peek();
                if (first == null)
                    available.await();
                
                else {
                    long delay = first.getDelay(NANOSECONDS);
                    
                    if (delay <= 0)
                        // 超时则出队
                        return q.poll();
                    
                    first = null; // don't retain ref while waiting
                    
                    if (leader != null)
                        
                        // 已经有线程等待这个元素，则当前线程无限阻塞
                        available.await();
                    else {
                        Thread thisThread = Thread.currentThread();
                        leader = thisThread;
                        try {
                            
                            // 没有其他线程等待这个元素，则有限阻塞
                            available.awaitNanos(delay);
                        } finally {
                            if (leader == thisThread)
                                leader = null;
                        }
                    }
                }
            }
        } finally {
            if (leader == null && q.peek() != null)
                
                // 当前线程是leader，则唤醒其他线程
                available.signal();
            lock.unlock();
        }
    }
```



# SynchronousQueue

> 本身没有容量

```java
public class SynchronousQueue<E> extends AbstractQueue<E>
    implements BlockingQueue<E>, java.io.Serializable {
	
    // 公平与非公平的实现
	public SynchronousQueue(boolean fair) {
        transferer = fair ? new TransferQueue<E>() : new TransferStack<E>();
    }
    
    public void put(E e) throws InterruptedException {
        if (e == null) throw new NullPointerException();
        if (transferer.transfer(e, false, 0) == null) {
            Thread.interrupted();
            throw new InterruptedException();
        }
    }
    
	public E take() throws InterruptedException {
        E e = transferer.transfer(null, false, 0);
        if (e != null)
            return e;
        Thread.interrupted();
        throw new InterruptedException();
    }
}
```



## TransferQueue的transfer方法

> TransferQueue为SynchronousQueue公平模式的实现

```java
		E transfer(E e, boolean timed, long nanos) {
            QNode s = null; 
            boolean isData = (e != null);

            for (;;) {
                QNode t = tail;
                QNode h = head;
                
                // 为初始化则重新开始循环
                if (t == null || h == null)         
                    continue;                     

                // 队列为空或模式相同，则进入if分支，阻塞
                if (h == t || t.isData == isData) {
                    QNode tn = t.next;
                    
                    // 不一致性读
                    if (t != tail)
                        continue;
                    if (tn != null) {
                        
                        // 后移指针
                        advanceTail(t, tn);
                        continue;
                    }
                    if (timed && nanos <= 0)        
                        return null;
                    if (s == null)
                        s = new QNode(e, isData);
                    if (!t.casNext(null, s))
                        
                        // cas失败则继续
                        continue;

                     // swing tail and wait
                    advanceTail(t, s);
                    Object x = awaitFulfill(s, e, timed, nanos);
                    
                    // wait was cancelled
                    if (x == s) {                   
                        clean(t, s);
                        return null;
                    }

                    if (!s.isOffList()) {           // not already unlinked
                        advanceHead(t, s);          // unlink if head
                        if (x != null)              // and forget fields
                            s.item = s;
                        s.waiter = null;
                    }
                    return (x != null) ? (E)x : e;

                } else {                           
                    
                    // 模式不同则进入else分支
                    QNode m = h.next;               // node to fulfill
                    if (t != tail || m == null || h != head)
                        continue;                   // inconsistent read

                    Object x = m.item;
                    if (isData == (x != null) ||    // 已经匹配过
                        x == m ||                   // m cancelled
                        !m.casItem(x, e)) {         // lost CAS
                        advanceHead(h, m);          // dequeue and retry
                        continue;
                    }

                    advanceHead(h, m);              // 匹配成功，出队
                    LockSupport.unpark(m.waiter);
                    return (x != null) ? (E)x : e;
                }
            }
        }
```



## TransferStack的transfer方法

> TransferStack为SynchronousQueue非公平模式的实现

```java
		E transfer(E e, boolean timed, long nanos) {

            SNode s = null; // constructed/reused as needed
            int mode = (e == null) ? REQUEST : DATA;

            for (;;) {
                SNode h = head;
                if (h == null || h.mode == mode) {  
                    if (timed && nanos <= 0) {      
                        if (h != null && h.isCancelled())
                            casHead(h, h.next);
                        else
                            return null;
                    } else if (casHead(h, s = snode(s, e, h, mode))) {
                        SNode m = awaitFulfill(s, timed, nanos);
                        if (m == s) {               
                            clean(s);
                            return null;
                        }
                        if ((h = head) != null && h.next == s)
                            casHead(h, s.next);     // help s's fulfiller
                        return (E) ((mode == REQUEST) ? m.item : s.item);
                    }
                    
                    // 尝试进行匹配
                } else if (!isFulfilling(h.mode)) {
                    if (h.isCancelled())            // already cancelled
                        casHead(h, h.next);         // pop and retry
                    
                    // 生成FULFILLING节点，入栈
                    else if (casHead(h, s=snode(s, e, h, FULFILLING|mode))) {
                        for (;;) { // loop until matched or waiters disappear
                            SNode m = s.next;       // m is s's match
                            if (m == null) {        // all waiters are gone
                                casHead(s, null);   // pop fulfill node
                                s = null;           // use new node next time
                                break;              // restart main loop
                            }
                            SNode mn = m.next;
                            if (m.tryMatch(s)) {
                                
                                // 2个元素一起出栈
                                casHead(s, mn);     
                                return (E) ((mode == REQUEST) ? m.item : s.item);
                            } else                  // lost match
                                s.casNext(m, mn);   // help unlink
                        }
                    }
                } else {                    
                    // 已经匹配过则出栈
                    SNode m = h.next;               // m is h's match
                    if (m == null)                  // waiter is gone
                        casHead(h, null);           // pop fulfilling node
                    else {
                        SNode mn = m.next;
                        if (m.tryMatch(h))          // help match
                            casHead(h, mn);         // pop both h and m
                        else                        // lost match
                            h.casNext(m, mn);       // help unlink
                    }
                }
            }
        }
```

# ConcurrentHashMap

> 相较于JDK7，JDK8进行了更好的处理
>
> 1. 使用了红黑树，查询速度更快
> 2. 加锁的粒度：对链表头节点加锁
> 3. 并发扩容，多个线程并发扩容
>
> 
>
> **关于sizeCtl的值**
>
> - `-1`：数组正在初始化
> - `- (n +1)`：n个线程正在进行扩容
> - `正数`：数组未初始化，则为数组容量，已经初始化，则为扩容阈值
> - `0`：数组未初始化，数组默认容量16
>
> 
>
> **数组初始化**
>
> 1. put方法才进行初始化
> 2. 容量为1.5倍初始容量 + 1，再向上取2的整次方
> 3. 设置sizeCtl
> 4. 对sizeCtl进行CAS设置为-1，线程发现为-1则让出CPU资源
>
> 
>
> **添加元素**
>
> 1. 对链表头节点使用synchronized，提高并发度
> 2. 记录数量
>
> 
>
> **扩容**
>
> 1. 创建2倍大小的数组
> 2. 设置任务步长（最小16）
> 3. CAS操作获取扩容任务，将范围内的原数组的链表移动到新数组
> 4. 原数组的链表将被拆分
> 5. 数组链表移动完成，则添加Forwarding节点，其他线程在原数组获取元素时，可以知道数组正在扩容，进行协助扩容

```java
public class ConcurrentHashMap<K,V> extends AbstractMap<K,V>
    implements ConcurrentMap<K,V>, Serializable {
    
    // 构造方法
        public ConcurrentHashMap(int initialCapacity) {
        if (initialCapacity < 0)
            throw new IllegalArgumentException();
        int cap = ((initialCapacity >= (MAXIMUM_CAPACITY >>> 1)) ?
                   MAXIMUM_CAPACITY :
                   
                   // 容量为1.5倍初始容量 + 1，再向上取2的整次方
                   tableSizeFor(initialCapacity + (initialCapacity >>> 1) + 1));
           
         // 初始化为容量
        this.sizeCtl = cap;
    }
}
```



## initTable方法

> 多线程竞争是通过对sizeCtl进行CAS操作实现的

```java
    private final Node<K,V>[] initTable() {
        Node<K,V>[] tab; int sc;
        while ((tab = table) == null || tab.length == 0) {
            if ((sc = sizeCtl) < 0)
                
                /* 
                 * 如果sizeCtl < 0，数组正在初始化，线程将循环执行
                 * 让出资源，知道数组初初始化完成
                 */
                Thread.yield(); 
            
            // CAS将sizeCtl改为-1
            else if (U.compareAndSwapInt(this, SIZECTL, sc, -1)) {
                try {
                    if ((tab = table) == null || tab.length == 0) {
                        int n = (sc > 0) ? sc : DEFAULT_CAPACITY;

                        Node<K,V>[] nt = (Node<K,V>[])new Node<?,?>[n];
                        table = tab = nt;
                        
                        // sizeCtl将被设置为下一次扩容的阈值，n-n/4
                        sc = n - (n >>> 2);
                    }
                } finally {
                    sizeCtl = sc;
                }
                break;
            }
        }
        return tab;
    }
```



## put方法

```java
    public V put(K key, V value) {
        return putVal(key, value, false);
    }

    final V putVal(K key, V value, boolean onlyIfAbsent) {
        if (key == null || value == null) throw new NullPointerException();
        int hash = spread(key.hashCode());
        int binCount = 0;
        for (Node<K,V>[] tab = table;;) {
            Node<K,V> f; int n, i, fh;
            if (tab == null || (n = tab.length) == 0)
                
                // 1. 数组初始化
                tab = initTable();
            else if ((f = tabAt(tab, i = (n - 1) & hash)) == null) {
                
                // 2. 数组中没有元素，则在数组添加节点
                if (casTabAt(tab, i, null,
                             new Node<K,V>(hash, key, value, null)))
                    break;                   // no lock when adding to empty bin
            }
            else if ((fh = f.hash) == MOVED)
                
                // 3.进行扩容
                tab = helpTransfer(tab, f);
            else {
                V oldVal = null;
                
                /*
                 * 4.数组上已经有元素，需要添加到链表中，
                 * 需要对链表头节点进行加锁，并发数为数组容量大小
                 */
                synchronized (f) {
                    if (tabAt(tab, i) == f) {
                        // 链表结构
                        if (fh >= 0) {
                            binCount = 1;
                            for (Node<K,V> e = f;; ++binCount) {
                                K ek;
                                if (e.hash == hash &&
                                    ((ek = e.key) == key ||
                                     (ek != null && key.equals(ek)))) {
                                    oldVal = e.val;
                                    if (!onlyIfAbsent)
                                        e.val = value;
                                    break;
                                }
                                Node<K,V> pred = e;
                                if ((e = e.next) == null) {
                                    pred.next = new Node<K,V>(hash, key,
                                                              value, null);
                                    break;
                                }
                            }
                        }
                        
                        // 添加到红黑树
                        else if (f instanceof TreeBin) {
                            Node<K,V> p;
                            binCount = 2;
                            if ((p = ((TreeBin<K,V>)f).putTreeVal(hash, key,
                                                           value)) != null) {
                                oldVal = p.val;
                                if (!onlyIfAbsent)
                                    p.val = value;
                            }
                        }
                    }
                }
                if (binCount != 0) {
                    if (binCount >= TREEIFY_THRESHOLD)
                        
                        /*
                         * 如果走链表分支，则会记录链表元素
                         * 数量都到阈值，就需要进行树化或扩容
                         */
                        treeifyBin(tab, i);
                    if (oldVal != null)
                        return oldVal;
                    break;
                }
            }
        }
        
        // 增加数量
        addCount(1L, binCount);
        return null;
    }
```

## treeifyBin方法

```java
    private final void treeifyBin(Node<K,V>[] tab, int index) {
        Node<K,V> b; int n, sc;
        if (tab != null) {
            
            if ((n = tab.length) < MIN_TREEIFY_CAPACITY)
                
                // 数组容量小于阈值64，进行扩容
                tryPresize(n << 1);
            
            else if ((b = tabAt(tab, index)) != null && b.hash >= 0) {
                
                // 转换为红黑树
                synchronized (b) {
                    if (tabAt(tab, index) == b) {
                        TreeNode<K,V> hd = null, tl = null;
                        for (Node<K,V> e = b; e != null; e = e.next) {
                            TreeNode<K,V> p =
                                new TreeNode<K,V>(e.hash, e.key, e.val,
                                                  null, null);
                            if ((p.prev = tl) == null)
                                hd = p;
                            else
                                tl.next = p;
                            tl = p;
                        }
                        setTabAt(tab, index, new TreeBin<K,V>(hd));
                    }
                }
            }
        }
    }
```

## 扩容数组与tryPresize调用的transfer方法

```java
    private final void transfer(Node<K,V>[] tab, Node<K,V>[] nextTab) {
        int n = tab.length, stride;
        
        //设置步长
        if ((stride = (NCPU > 1) ? (n >>> 3) / NCPU : n) < MIN_TRANSFER_STRIDE)
            stride = MIN_TRANSFER_STRIDE; // subdivide range
        
        // 数组将进行初始化
        if (nextTab == null) {            
            try {

                // 创建2倍容量的数组
                Node<K,V>[] nt = (Node<K,V>[])new Node<?,?>[n << 1];
                nextTab = nt;
            } catch (Throwable ex) {     
                sizeCtl = Integer.MAX_VALUE;
                return;
            }
            
            //新数组
            nextTable = nextTab;
            
            // 初始化为旧数组长度
            transferIndex = n;
        }
        int nextn = nextTab.length;
        ForwardingNode<K,V> fwd = new ForwardingNode<K,V>(nextTab);
        boolean advance = true;
        boolean finishing = false; 
        for (int i = 0, bound = 0;;) {
            Node<K,V> f; int fh;
            while (advance) {
                int nextIndex, nextBound;
                if (--i >= bound || finishing)
                    advance = false;
                
                // transferIndex <= 0，扩容结束
                    else if ((nextIndex = transferIndex) <= 0) {
                    i = -1;
                    advance = false;
                }
                
                // CAS操作，线程获取扩容任务的范围
                else if (U.compareAndSwapInt
                         (this, TRANSFERINDEX, nextIndex,
                          nextBound = (nextIndex > stride ?
                                       nextIndex - stride : 0))) {
                    
                    /*
                     * 进行扩容时，需要将旧数组的元素移动到新数组，
                     * 多个线程可以同时处理不同的范围，而bound是
                     * 某个范围的最小下标
                     */
                    bound = nextBound;
                    
                    /*
                     * i为范围最大下标，i--，从范围的后方向前遍历，将元素
                     * 移动到新数组
                     */
                    i = nextIndex - 1;
                    
                    advance = false;
                }
            }
            
            // i < 0，当前线程完成最后的扩容任务
            if (i < 0 || i >= n || i + n >= nextn) {
                int sc;
                
                if (finishing) {
                    nextTable = null;
                    
                    // 完成扩容，更新信息，设置新数组和sizeCtl
                    table = nextTab;
                    sizeCtl = (n << 1) - (n >>> 1);
                    return;
                }
                if (U.compareAndSwapInt(this, SIZECTL, sc = sizeCtl, sc - 1)) {
                    if ((sc - 2) != resizeStamp(n) << RESIZE_STAMP_SHIFT)
                        return;
                    finishing = advance = true;
                    i = n; // recheck before commit
                }
            }
            else if ((f = tabAt(tab, i)) == null)
                
                // 数组某个位置迁移完成，则添加FWD节点
                advance = casTabAt(tab, i, null, fwd);
            else if ((fh = f.hash) == MOVED)
                advance = true; // already processed
            else {
                synchronized (f) {
                    if (tabAt(tab, i) == f) {
                        Node<K,V> ln, hn;
                        if (fh >= 0) {
                            int runBit = fh & n;
                            Node<K,V> lastRun = f;
                            for (Node<K,V> p = f.next; p != null; p = p.next) {
                                int b = p.hash & n;
                                if (b != runBit) {
                                    runBit = b;
                                    lastRun = p;
                                }
                            }
                            if (runBit == 0) {
                                ln = lastRun;
                                hn = null;
                            }
                            else {
                                hn = lastRun;
                                ln = null;
                            }
                            for (Node<K,V> p = f; p != lastRun; p = p.next) {
                                int ph = p.hash; K pk = p.key; V pv = p.val;
                                
                                // 构成不同的链表
                                if ((ph & n) == 0)
                                    ln = new Node<K,V>(ph, pk, pv, ln);
                                else
                                    hn = new Node<K,V>(ph, pk, pv, hn);
                            }
                            
                            
                            /*
                             * 原数组上的一条链表的元素只会放到新数组
                             * 上的2个位置，即 i、I + 旧数组长度
                             *
                             * 原因：节点的放入位置计算方式是 key的哈希值 & 数组长度 -1，
                             * 而数组长度是2的次幂，如：
                             * key哈希值 & 01111（16-1）
                             * key哈希值 & 11111（32-1）
                             * 
                             * 下方setTabAt方法将原一条链表，分成2部分，插入到
                             * 新数组不同位置
                             */
                            setTabAt(nextTab, i, ln);
                            setTabAt(nextTab, i + n, hn);
                            
                            setTabAt(tab, i, fwd);
                            advance = true;
                        }
                        else if (f instanceof TreeBin) {
                            TreeBin<K,V> t = (TreeBin<K,V>)f;
                            TreeNode<K,V> lo = null, loTail = null;
                            TreeNode<K,V> hi = null, hiTail = null;
                            int lc = 0, hc = 0;
                            for (Node<K,V> e = t.first; e != null; e = e.next) {
                                int h = e.hash;
                                TreeNode<K,V> p = new TreeNode<K,V>
                                    (h, e.key, e.val, null, null);
                                if ((h & n) == 0) {
                                    if ((p.prev = loTail) == null)
                                        lo = p;
                                    else
                                        loTail.next = p;
                                    loTail = p;
                                    ++lc;
                                }
                                else {
                                    if ((p.prev = hiTail) == null)
                                        hi = p;
                                    else
                                        hiTail.next = p;
                                    hiTail = p;
                                    ++hc;
                                }
                            }
                            ln = (lc <= UNTREEIFY_THRESHOLD) ? untreeify(lo) :
                                (hc != 0) ? new TreeBin<K,V>(lo) : t;
                            hn = (hc <= UNTREEIFY_THRESHOLD) ? untreeify(hi) :
                                (lc != 0) ? new TreeBin<K,V>(hi) : t;
                            setTabAt(nextTab, i, ln);
                            setTabAt(nextTab, i + n, hn);
                            setTabAt(tab, i, fwd);
                            advance = true;
                        }
                    }
                }
            }
        }
    }
```

## tryPresize方法

```java
    private final void tryPresize(int size) {
        
        /* treeifyBin调用tryPresize(n << 1)
         * c被设置位 size的1.5倍向上取2的次幂
         */
        int c = (size >= (MAXIMUM_CAPACITY >>> 1)) ? MAXIMUM_CAPACITY :
            tableSizeFor(size + (size >>> 1) + 1);
        int sc;
        while ((sc = sizeCtl) >= 0) {
            Node<K,V>[] tab = table; int n;
            
            
            // 哈希表初始化
            if (tab == null || (n = tab.length) == 0) {
                n = (sc > c) ? sc : c;
                if (U.compareAndSwapInt(this, SIZECTL, sc, -1)) {
                    try {
                        if (table == tab) {
                            @SuppressWarnings("unchecked")
                            Node<K,V>[] nt = (Node<K,V>[])new Node<?,?>[n];
                            table = nt;
                            
                            // 扩容阈值
                            sc = n - (n >>> 2);
                        }
                    } finally {
                        sizeCtl = sc;
                    }
                }
            }
            else if (c <= sc || n >= MAXIMUM_CAPACITY)
                break;
            else if (tab == table) {
                int rs = resizeStamp(n);
                
                // 线程正在扩容
                if (sc < 0) {
                    Node<K,V>[] nt;
                    
                    // 扩容结束
                    if ((sc >>> RESIZE_STAMP_SHIFT) != rs || sc == rs + 1 ||
                        sc == rs + MAX_RESIZERS || (nt = nextTable) == null ||
                        transferIndex <= 0)
                        break;
                    
                    if (U.compareAndSwapInt(this, SIZECTL, sc, sc + 1))
                        
                        // 线程帮忙扩容
                        transfer(tab, nt);
                }
                else if (U.compareAndSwapInt(this, SIZECTL, sc,
                                             (rs << RESIZE_STAMP_SHIFT) + 2))
                    
                    // 第一次扩容
                    transfer(tab, null);
            }
        }
    }
```
